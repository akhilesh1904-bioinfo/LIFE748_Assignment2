# Part 2 - Machine Learning Analysis of Differential Expression

This folder contains the code and selected outputs for Part 2 of LIFE748 Assessment 2.

## Files
- `Part2_code_cleaned.qmd` - Quarto workflow for DEG analysis and machine learning
- `figures/pca_plot.png` - PCA plot
- `figures/heatmap.png` - Heatmap of differentially expressed genes

## Analysis included
- DESeq2 differential expression analysis
- Variance stabilising transformation
- DEG filtering
- PCA
- K-means clustering
- Hierarchical clustering heatmap
- Logistic regression
- LDA
- SVM
- Cross-validation extension
